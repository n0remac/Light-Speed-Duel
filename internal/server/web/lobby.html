Light Speed Duel

This is a 2 player dogfighting game set across lightyears of space.

Flight Notes

  • Time delay and time dilation are core mechanics of this game. When you see an enemy ship, you are actually seeing it as it was in the past.
  • The further away a ship is, the more the offset in its actual position will be.
  • You can use this information to help you lead shots when planning missile routes.
  • Time dilation is expressed though cool down times. When you are moving faster missile cool downs will be longer.

Game Modes

Story-driven experience with tutorial and narrative elements.

Call Sign

Your call sign is stored locally and sent to the server when you join a room.

Map Size

Larger maps allow for more strategic gameplay and longer battles. Use zoom controls (+/- or mouse wheel) to navigate.
